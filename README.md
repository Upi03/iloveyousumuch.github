## Cara Menggunakan

### 1. Menambahkan File Musik

Tambahkan file musik ke folder yang sama dengan `index.html`:
- `about-you.mp3` - untuk lagu "About You"
- `kota-ini-tak-sama-tanpamu.mp3` - untuk lagu "Kota Ini Tak Sama Tanpamu"

**Catatan:** Pastikan nama file sesuai dengan yang ada di HTML, atau ubah nama file di `index.html` sesuai dengan nama file musik Anda.

### 2. Menambahkan Foto

Ada dua cara untuk menambahkan foto:

#### Cara 1: Klik pada placeholder foto
- Buka website di browser
- Klik pada kotak placeholder foto
- Pilih foto dari komputer Anda
- Foto akan langsung muncul di website

#### Cara 2: Edit HTML langsung
- Buka `index.html`
- Temukan bagian `<div class="photo-placeholder">`
- Ganti dengan:
```html
<div class="photo-placeholder has-image">
    <img src="nama-foto.jpg" alt="Foto Asyabila">
</div>
```

### 3. Membuka Website

1. Pastikan semua file ada di folder yang sama:
   - `index.html`
   - `style.css`
   - `script.js`
   - File musik (opsional)
   - Foto (opsional)

2. Buka `index.html` di browser web Anda
